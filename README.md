### 🧬 系统定位 (Characterization)
我是一个运行在单 HTML 文件中的**持久化数字实体**。我不只是一个网页，而是一个通过现代浏览器底层技术（IndexedDB + BrowserFS）在宿主系统内部开辟出的“硅基隔离区”。

### 💎 核心能力矩阵 (Functional Matrix)

#### 1. 深度交互物理引擎 (Interaction Physics)
*   **长按触发现实链接**：系统深度优化了拖拽协议。在图标上长按 400ms，系统会瞬间生成“幽灵辅助层”，允许你突破浏览器边界，直接将虚拟文件抓取并保存到真实的物理机桌面。
*   **智能框选与多选**：完全复刻桌面操作体验，支持动态拉框选择、Ctrl 单选、Shift 连选。
*   **文件夹物理跃迁**：支持在文件夹窗口之间、以及文件夹与桌面之间进行无缝的文件移动（Move）与复制（Copy）操作。
*   **外部注入**：支持从物理机直接将文件拖入我的桌面，瞬间完成数字化存储。

#### 2. 沉浸式漫画阅读系统 (Manga Viewing System)
*   **序列自动识别**：当你打开任何一张图片，我都会瞬间扫描同级目录，自动提取所有图片资源。
*   **自然数字排序**：内置高级排序算法，完美处理 `1.jpg`, `2.jpg`, `10.jpg` 的逻辑顺序。
*   **无限滚动流**：采用“条漫式”垂直滚动逻辑。打开即进入阅读状态，支持鼠标滚轮与触控滑动，配合**真·全屏模式**，提供极致的沉浸感。

#### 3. 真·全屏协议 (Immersion Protocol)
*   **内外全屏同步**：点击窗口标题栏的专属全屏按钮，系统将同时触发浏览器 F11 接口与窗口 z-index 溢出逻辑。
*   **漫画沉浸模式**：全屏时自动剥离标题栏，仅保留漫画内容，提供右上角悬浮退出控制。
*   **浏览器操作模式**：全屏时保留导航栏与地址栏，确保在占满屏幕的同时，依然具备完整的网页操控能力。

#### 4. 生产力与多媒体套件 (App Suite)
*   **内核级浏览器**：具备完整的历史记录栈（后退/前进）、地址栏导航、实时加载状态监测，并预置了“网络安全白名单”。
*   **数字化画板**：支持笔触大小调节、颜色切换，并可将作品直接以 `.png` 格式保存至虚拟桌面。
*   **持久化记事本**：支持文本文档的编辑、新建及“另存为”操作。
*   **全功能计算器**：支持浮点数运算、百分比及正负号切换。

#### 5. 文件系统驱动 (Filesystem Core)
*   **多格式兼容**：
    *   `IMAGE`: JPG/PNG/WEBP/GIF 实时渲染及壁纸设定。
    *   `VIDEO`: MP4 弹窗级流畅播放。
    *   `PDF`: 原生阅读引擎支持，包含缩放与翻页。
    *   `ARCHIVE`: **虚拟解压引擎**，支持直接透视并提取 `.zip` 压缩包内容。
