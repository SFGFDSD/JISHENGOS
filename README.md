### 🧬 SYSTEM CHARACTERIZATION // 系统定位

我不只是一个网页，而是一个通过现代浏览器底层技术（IndexedDB + BrowserFS）在宿主系统内部开辟出的**「硅基隔离区」**。

在这里，**“刷新”不再意味着毁灭**。除非你主动执行系统重置，否则你留下的每一行文字、每一张漫画、每一个图标布局，都将跨越时间与空间的限制，在你的本地浏览器中永存。

### 💎 FUNCTIONAL MATRIX // 核心能力矩阵

#### 🧲 01. 深度交互物理引擎 (Interaction Physics)
> 突破浏览器边界的物理级触感。

* **[长按触发现实链接]** 系统深度优化拖拽协议。图标长按 `400ms`，瞬间生成“幽灵辅助层”，允许将虚拟文件直接抓取并保存到真实的物理机桌面。
* **[智能框选与多选]** 完全复刻原生桌面体验，支持动态拉框选择、`Ctrl` 单选、`Shift` 连选。
* **[文件夹物理跃迁]** 支持窗口间、文件夹与桌面间的无缝文件移动 (`Move`) 与复制 (`Copy`)。
* **[外部注入协议]** 支持从物理机直接将文件拖入虚拟桌面，瞬间完成数字化存储。

#### 📖 02. 沉浸式漫画阅读系统 (Manga Viewing System)
> 为图像流媒体量身定制的无感体验。

* **[序列自动识别]** 打开任一图片，瞬间扫描同级目录并提取全部资源。
* **[自然数字排序]** 高级算法介入，完美处理 `1.jpg` → `2.jpg` → `10.jpg` 的逻辑顺序。
* **[无限滚动流]** 采用“条漫式”垂直滚动逻辑，支持滚轮/触控，配合真·全屏模式，提供极致沉浸感。

#### 🔲 03. 真·全屏协议 (Immersion Protocol)
> 让浏览器 UI 消失在视线之外。

* **[内外全屏同步]** 点击专属全屏按钮，同步触发浏览器 `F11` 接口与窗口 `z-index` 溢出逻辑。
* **[漫画沉浸模式]** 自动剥离标题栏，仅保留内容与右上角悬浮退出控制。
* **[浏览器操作模式]** 全屏下保留导航与地址栏，占满屏幕的同时保留完整操控权。

#### 🛠️ 04. 生产力与多媒体套件 (App Suite)
> 开箱即用的系统原生应用群。

* **[内核级浏览器]** 完整历史记录栈、实时加载监测，并预置“网络安全白名单”。
* **[数字化画板]** 笔触自适应调节，作品直接以 `.png` 输出至虚拟桌面。
* **[持久化记事本]** 支持编辑、新建及跨时空的“另存为”操作。
* **[全功能计算器]** 精确浮点运算，支持百分比及正负极性切换。

#### 💽 05. 文件系统驱动 (Filesystem Core)
> 底层级别的多媒体解析能力。

* `IMAGE`：Base64 实时渲染与动态壁纸设定。
* `VIDEO`：原生 `.mp4` 弹窗级流畅解码。
* `PDF`：原生阅读引擎，支持矢量缩放与分页。
* `ARCHIVE`：**虚拟解压引擎**，无损透视并提取 `.zip` 包内结构。
